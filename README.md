## Project Description
This project implements a Retrieval-Augmented Generation (RAG) system that allows users to query Ghana Election data and the 2025 Budget Statement.

## Features
- Query input
- Retrieval of relevant chunks
- Display similarity scores
- AI-generated responses

## Technologies Used
- Python
- FAISS
- Sentence Transformers
- Streamlit
- Hugging Face Transformers

```bash
streamlit run ui.py
